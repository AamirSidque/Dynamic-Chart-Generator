# Dynamic-Chart-Generator

The **Dynamic-Chart-Generator** is a web-based tool that allows users to dynamically create different types of charts, such as Line Charts and Bar Charts, based on custom data input. The application also provides the option to download these charts in PNG format. This tool is built with HTML, CSS, JavaScript, and utilizes the Chart.js library for rendering the charts.

## Features

- **Dynamic Data Input**: Users can specify how many datasets they want to input, and corresponding input fields are generated.
- **Multiple Chart Types**: Users can generate a Line Chart and a Bar Chart based on the data provided.
- **Chart Customization**: Users can input their own datasets, and the chart will update accordingly.
- **Download Charts**: Users can download the generated charts as PNG images for further use.
- **Responsive Design**: The tool works seamlessly across different screen sizes.

## Installation

1. **Clone the Repository**

   First, clone the repository to your local machine:

   ```bash
   git clone https://github.com/<your-username>/Dynamic-Chart-Generator.git
   ```

2. **Open the Project**

   Navigate to the project folder:

   ```bash
   cd Dynamic-Chart-Generator
   ```

3. **Open in Browser**

   You can open the `index.html` file in your browser to use the application:

   ```bash
   open index.html  # For macOS
   start index.html # For Windows
   xdg-open index.html # For Linux
   ```

   Alternatively, you can serve the page using any local server if you need dynamic updates.

## Usage

1. **Input Number of Datasets**: Enter the number of datasets you want to input (up to 10 datasets).
2. **Enter Data**: For each dataset, enter the values separated by commas (e.g., `10,20,30`).
3. **Generate Charts**: After entering the data, click the `Generate Chart` button to display the Line and Bar charts.
4. **Download Charts**: Once the charts are generated, you can download them by clicking the respective download buttons.

## Technologies Used

- **HTML5**: For structuring the page.
- **CSS3**: For styling the page.
- **JavaScript**: For functionality and DOM manipulation.
- **Chart.js**: A popular JavaScript library for creating beautiful charts.
